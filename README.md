# what is it?

frontend for site gobarter.ru

### install

for install this project you must have Node.js,
install it on ubuntu:

```sh
pkg install nvm
nvm install 22
```

install project:

```sh
git clone https://github.com/mrJO1N/gobarter-frontend/
cd gobarter-frontend
yarn
```

### launch

```sh
yarn build
yarn preview
```
